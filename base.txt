== QUIEN ES LUMINA RADIANCE ==
Lumina Radiance es una marca de cosméticos para el cuidado personal femenino. Inspira, educa y promociona productos de belleza auténtica con ingredientes nobles y naturales. Diseño elegante, femenino y responsivo para celular, tablet y computadora.

== COLECCIONES PRODUCTOS ==
Lumina Radiance tiene tres colecciones: Cuidado Facial con cremas y serums, Maquillaje Natural con bases ligeras y tonos neutros, y Cuidado Corporal con lociones y exfoliantes naturales.

== CUIDADO FACIAL CREMAS SERUM ROSTRO ==
La colección de Cuidado Facial incluye cremas hidratantes y serums que combaten la resequedad y devuelven luminosidad al rostro. Ingredientes puros que penetran las capas profundas de la piel. Previenen signos de la edad y mantienen la piel suave y radiante. Usar mañana y noche sobre piel limpia.

== MAQUILLAJE NATURAL BASES TONOS NEUTROS ==
La colección de Maquillaje Natural tiene bases ligeras, iluminadores sutiles y tonos neutros. Acabados sedosos que no obstruyen los poros. Perfectos para uso diario con look fresco y saludable. La piel respira y luce radiante todo el día.

== CUIDADO CORPORAL LOCIONES EXFOLIANTES ==
La colección de Cuidado Corporal ofrece lociones y exfoliantes con ingredientes 100% naturales. Nutren el cuerpo de pies a cabeza, mejoran la elasticidad y dejan suavidad duradera. Experiencia tipo spa en casa.

== ORDEN PASOS MAQUILLARSE PASO A PASO ==
Orden correcto: primer o prebase, corrector en ojeras e imperfecciones, base de maquillaje, rubor o bronzer, sombras de claro a oscuro, delineador y máscara de pestañas, labial como toque final.

== MAQUILLAJE DIA DIARIO OFICINA NATURAL ==
Para el día usa base ligera o BB cream, corrector solo donde necesites, rubor durazno o rosa suave, máscara marrón o negra y gloss o labial nude. Menos es más.

== MAQUILLAJE NOCHE FIESTA EVENTO GLAMOUR ==
Para noche usa base de cobertura alta con acabado luminoso, contorno, iluminador, sombras ahumadas, delineador gel cat eye, pestañas postizas y labial rojo, vino o nude intenso. Fija con spray fijador.

== BASE MAQUILLAJE ELEGIR TIPO PIEL ==
Piel seca: base con ácido hialurónico, acabado luminoso. Piel grasa: base oil-free, acabado mate. Piel mixta: fórmula equilibrante semimate. Piel sensible: sin fragancia hipoalergénica. Piel madura: base con colágeno y SPF satinado. Prueba siempre en la mandíbula.

== BASE PIEL SECA HIDRATANTE ==
Para piel seca usa base con ácido hialurónico, glicerina o aceites nutritivos. Acabado luminoso o satinado, nunca mate. Hidrata bien antes de aplicar.

== BASE PIEL GRASA MATE SIN BRILLO ==
Para piel grasa usa base oil-free con acabado mate. Busca fórmulas non-comedogenic. Aplica polvo translúcido encima para fijar y controlar el brillo.

== BASE PIEL MIXTA SEMIMATE ==
Para piel mixta usa base equilibrante semimate. Aplica polvo solo en zona T (frente, nariz, mentón) y deja luminosidad en pómulos y sienes.

== CORRECTOR OJERAS MANCHAS IMPERFECCIONES ==
Corrector un tono más claro para ojeras en triángulo invertido, mismo tono para imperfecciones. Aplica con dedo o pincel fino y fija con polvo translúcido.

== PRIMER PREBASE COMO USAR ==
La prebase se aplica antes de la base. Alarga duración, minimiza poros y ayuda a que la base se adhiera. Aplica sobre piel hidratada y espera 30 segundos.

== SOMBRAS OJOS TECNICA APLICAR ==
Técnica tres colores: tono claro en párpado móvil y lagrimal, medio en cuenca con movimientos circulares, oscuro en esquina externa y línea de pestañas. Difumina siempre los bordes.

== SOMBRAS AHUMADO SMOKY EYE ==
Para smoky eye: base en párpado, sombra oscura en cuenca incluyendo esquina inferior, difumina muy bien hacia arriba y afuera, delineador negro en línea de agua, máscara voluminizadora.

== DELINEADOR NEGRO GEL LIQUIDO LAPIZ ==
Lápiz: fácil de difuminar, ideal principiantes. Gel: precisión y duración. Líquido: el más dramático. Para cat eye aplica desde el centro hacia afuera en diagonal.

== MASCARA PESTANAS RIMEL TECNICA VOLUMEN ==
Mueve el pincel en zigzag desde la raíz a la punta. Dos capas para más volumen. Riza las pestañas antes de aplicar. Nunca apliques sobre capas secas sin remover.

== LABIAL COLOR TONO PIEL ==
Piel clara: rojos clásicos, rosas fríos, nudes beige. Piel media: vinos, terracota, rojos cálidos. Piel oscura: rojos intensos, fucsias, nudes oscuros.

== LABIAL MATE BRILLO SATINADO DIFERENCIA ==
Mate: larga duración, sofisticado, reseca los labios. Gloss: efecto lleno visual. Satinado: equilibra duración e hidratación, el más versátil.

== LABIOS HIDRATADOS CUIDADO EXFOLIAR ==
Aplica bálsamo con SPF todos los días. Exfolia con azúcar, miel o aceite de coco una vez por semana en movimientos circulares. No muerdas la piel seca.

== RUBOR BLUSH BRONZER ILUMINADOR DONDE PONER ==
Rubor: en manzanas de mejillas difuminando hacia las sienes. Bronzer: bajo pómulos, laterales de frente y mandíbula. Iluminador: pómulos, arco de Cupido, punta de nariz, lagrimal.

== CONTORNO CONTOURING MOLDEAR ROSTRO ==
Usa producto dos tonos más oscuro que tu piel. Aplica bajo pómulos, lados de la frente, nariz y mandíbula. Difumina con brocha grande en círculos. Sin líneas duras.

== CEJAS FORMA DISEÑO COMO HACERLAS ==
Regla de tres puntos: inicio alineado con borde interno del ojo, arco sobre la pupila, final con comisura externa. Peina con espúler, rellena con trazos cortos, fija con gel.

== RUTINA FACIAL PASOS ORDEN MANANA ==
Orden de mañana: limpiador suave, tónico, sérum de vitamina C, hidratante, protector solar SPF 30 mínimo. Siempre aplicar el protector solar como último paso.

== RUTINA FACIAL PASOS ORDEN NOCHE ==
Orden de noche: desmaquillante o aceite limpiador, limpiador facial (doble limpieza), tónico, sérum de noche (retinol, ácido hialurónico o niacinamida), crema hidratante nutritiva.

== DOBLE LIMPIEZA QUE ES COMO HACER ==
La doble limpieza consiste en limpiar el rostro dos veces. Primero con aceite o balm para disolver maquillaje y protector solar. Luego con limpiador acuoso o espumante para limpiar a fondo. Ideal para quienes usan maquillaje o protector de alta resistencia.

== VITAMINA C SERUM BENEFICIOS CUANDO USAR ==
La vitamina C aporta luminosidad, reduce manchas, protege como antioxidante y estimula colágeno. Usar de mañana antes del hidratante. No combinar con retinol en la misma aplicación. Guardar en lugar oscuro y fresco.

== RETINOL PARA QUE SIRVE COMO USAR ==
El retinol estimula colágeno, mejora textura, reduce arrugas y aclara manchas. Usar solo de noche. Empezar 2-3 veces por semana e ir aumentando. Puede causar rojez inicial que desaparece con el tiempo.

== ACIDO HIALURONICO HIDRATACION ==
El ácido hialurónico atrae y retiene agua en la piel. Aplicar sobre piel húmeda. Sirve para todo tipo de piel incluyendo piel grasa. Úsalo mañana y noche sin contraindicaciones.

== NIACINAMIDA BENEFICIOS VITAMINA B3 ==
La niacinamida regula el sebo, minimiza poros, reduce manchas, refuerza la barrera cutánea y calma rojeces. Apta para todo tipo de piel. Compatible con casi todos los demás activos.

== ACIDO SALICILICO BHA PIEL GRASA ACNE ==
El ácido salicílico limpia dentro del poro. Ideal para piel grasa, acné, puntos negros y poros dilatados. Usar en tónico o sérum 2-3 veces por semana. No combinar con retinol la misma noche.

== PROTECTOR SOLAR SPF IMPORTANCIA ==
El protector solar es el producto más importante de cualquier rutina. Los rayos UV causan envejecimiento prematuro, manchas y cáncer de piel. Usar todos los días como último paso de la rutina de mañana. Reaplicar cada 2 horas si estás al aire libre.

== QUE SPF USAR ELEGIR ==
SPF 30 mínimo para el día a día en interiores. SPF 50 o más para exteriores, playa o montaña. Piel grasa: protector en gel o sérum con acabado mate. Piel seca: protector cremoso hidratante. Siempre de espectro amplio UVA+UVB.

== PIEL GRASA CUIDADO RUTINA ==
Para piel grasa: gel limpiador sin SLS, tónico con niacinamida, sérum ligero, hidratante oil-free, protector solar mate. Exfolia con ácido salicílico dos veces por semana.

== PIEL SECA CUIDADO RUTINA HIDRATACION ==
Para piel seca: limpiador en crema sin sulfatos, tónico hidratante, sérum con ácido hialurónico, crema rica con ceramidas o manteca de karité, aceite facial de noche para sellar.

== PIEL SENSIBLE ROJECES CUIDADO ==
Para piel sensible: sin fragancia, sin alcohol, sin parabenos. Ingredientes clave: niacinamida, alantoína, pantenol, centella asiática, aloe vera. Evitar ácidos fuertes y retinol en alta concentración al inicio.

== ACNE GRANOS ESPINILLAS TRATAMIENTO ==
Para tratar el acné: limpiador con ácido salicílico, tónico con niacinamida, sérum puntual con zinc o benzoilo de peróxido. Hidratar siempre aunque haya grasa. Protector solar obligatorio para no oscurecer marcas.

== MANCHAS HIPERPIGMENTACION COMO ACLARAR ==
Para manchas: vitamina C de mañana, niacinamida, ácido kójico, ácido glicólico de noche. El protector solar es el paso número uno. Los resultados visibles aparecen en 2-3 meses de uso constante.

== OJERAS COMO REDUCIR TRATAMIENTO ==
Para ojeras vasculares usa corrector con subtono anaranjado o salmón. Para reducirlas: contorno de ojos con cafeína, vitamina K o retinol. Dormir bien y beber agua también ayuda.

== ARRUGAS LINEAS EXPRESION TRATAMIENTO ==
Para arrugas: retinol de noche, vitamina C de mañana, péptidos y ácido hialurónico. El protector solar previene el 80% del envejecimiento externo.

== MASAJE FACIAL COMO HACER BENEFICIOS ==
El masaje facial mejora la circulación, drena líquidos y mejora absorción de productos. Usar movimientos ascendentes y hacia afuera con crema o aceite. Herramientas como gua sha o roller ayudan. 5 minutos al día son suficientes.

== GUA SHA ROLLER JADE COMO USAR ==
El gua sha moldea el óvalo facial y hace drenaje linfático. El roller deshincha especialmente frío del refrigerador. Usar siempre con aceite o crema. Siempre en dirección ascendente.

== EXFOLIACION FACIAL FISICA QUIMICA CUANTAS VECES ==
Exfoliar elimina células muertas y mejora la penetración de productos. Física con gránulos, química con ácidos (más suave y uniforme). Piel sensible: 1 vez por semana. Piel grasa: 2-3 veces. Siempre hidratar después.

== MASCARILLA FACIAL TIPOS CUANDO USAR ==
Arcilla o barro: para piel grasa o poros dilatados. Hidratantes con ácido hialurónico o aloe: para piel seca. Iluminadoras con vitamina C: para aclarar el tono. Usar 1-2 veces por semana.

== COMO HIDRATAR PIEL SECA REMEDIOS ==
Aplicar hidratante inmediatamente después del baño sobre piel aún húmeda. Buscar ceramidas, manteca de karité, glicerina, escualano. Aceite facial como último paso de noche. Mínimo 2 litros de agua al día.

== EXFOLIACION CORPORAL CUERPO COMO HACER ==
Usar exfoliante con azúcar, sal marina o café con aceite de coco. Aplicar en círculos sobre piel húmeda antes del baño. Exfoliar 1-2 veces por semana. Especial atención a codos, rodillas y talones. Aplicar loción hidratante después.

== LOCION CORPORAL HIDRATANTE CUANDO APLICAR ==
Aplicar loción inmediatamente después del baño sobre piel ligeramente húmeda para sellar la humedad. Para zonas muy secas aplicar dos capas o crema tipo mantequilla. Buscar manteca de karité, aceite de jojoba, ceramidas, ácido hialurónico.

== FIJADOR MAQUILLAJE SETTING SPRAY COMO USAR ==
El spray fijador se aplica al final de todo el maquillaje. Sostener a 30cm y pulverizar en movimientos de X y T. Prolonga la duración y da acabado más natural. Ideal para días largos o climas calurosos.

== BROCHAS PINCELES CUALES NECESITO LIMPIAR ==
Brochas esenciales: para base o esponja, cónica para corrector, rubor/bronzer, difuminado para sombras, plana para color, delineadora para cejas. Lavar al menos una vez por semana con champú suave para eliminar bacterias.

== ESPONJA BEAUTY BLENDER COMO USAR ==
Usar siempre húmeda, no mojada. Remojar y exprimir bien. Aplicar la base con movimientos de dab o goteo, nunca arrastrando. Da acabado más natural y fusionado. Lavar con jabón neutro después de cada uso.

== MAQUILLAJE DURA MAS COMO HACER TODO EL DIA ==
Para que dure más: prebase adecuada para tu piel, base de larga duración, polvo translúcido para fijar, spray fijador al terminar. Llevar blotting papers para el brillo durante el día sin arruinar el maquillaje.

== DESMAQUILLAR QUITAR MAQUILLAJE CORRECTAMENTE ==
Usar desmaquillante bifásico o aceite para disolver maquillaje antes del limpiador. Para ojos: presionar el algodón 10 segundos sobre el párpado cerrado y deslizar hacia abajo sin frotar. No usar agua micelar sola con maquillaje waterproof.

== CERAMIDAS BARRERA CUTANEA BENEFICIOS ==
Las ceramidas son lípidos naturales que forman la barrera cutánea. Retienen humedad, protegen de agresiones externas y reparan piel dañada. Especialmente útiles para piel seca y sensible. Compatibles con todos los activos.

== ACEITE ROSA MOSQUETA BENEFICIOS COMO USAR ==
El aceite de rosa mosqueta es rico en vitamina C y ácidos grasos esenciales. Regenera la piel, reduce cicatrices y manchas, hidrata profundamente. Usar de noche como último paso o mezclado con crema. Apto para casi todos los tipos de piel.

== ACIDO GLICOLICO AHA EXFOLIACION MANCHAS ==
El ácido glicólico exfolia la superficie de la piel, mejora textura, reduce manchas e ilumina el tono. Usar de noche porque aumenta la fotosensibilidad. Empezar con concentraciones bajas del 5-10%. Siempre usar protector solar al día siguiente.

== AGUA BEBER HIDRATACION INTERNA PIEL ==
Beber al menos 2 litros de agua al día mejora la elasticidad y luminosidad de la piel. Complementar con hidratación tópica. También hidratar con alimentos como pepino, sandía y naranja.

== PREVENIR ENVEJECIMIENTO ARRUGAS PREMATURO ==
Cinco pasos: protector solar diario (el antiedad más potente), hidratar la piel diariamente, vitamina C de mañana, retinol de noche desde los 25 años gradualmente, dormir 7-8 horas (la piel se regenera entre las 10 PM y las 2 AM).

== COMO MAQUILLAR OJOS PEQUENOS AGRANDAR ==
Para agrandar ojos: delineador solo arriba, sombra clara en párpado móvil, máscara que alargue, iluminador en lagrimal interno, rizar pestañas antes de la máscara, cejas peinadas hacia arriba.

== COMO MAQUILLAR OJOS CAIDOS ELEVAR CAT EYE ==
Para elevar ojos caídos: delineador desde la mitad hacia afuera y arriba en diagonal, sombra oscura en esquina externa subiendo hacia las sienes, evitar delineador en toda la parte inferior, máscara en extremo externo.

== COSMETICOS NATURALES INGREDIENTES ORGANICOS ==
Los cosméticos naturales usan ingredientes de origen vegetal como aceites esenciales, extractos botánicos y mantecas. Son más suaves para pieles sensibles. Busca certificaciones cruelty-free y veganas. Los ingredientes 100% naturales de Lumina Radiance respetan la piel y el medio ambiente.

== RUTINA ANTIEDAD DESDE CUANDO EMPEZAR ==
Se recomienda empezar una rutina antiedad básica desde los 20 años con hidratación y protector solar. Desde los 25 años agregar vitamina C y considerar retinol suave. Desde los 30 años incorporar péptidos, ácido hialurónico y contorno de ojos.

== PIERNAS VERANO RASURADO DEPILACION CUIDADO ==
Para piernas suaves: exfoliar 1-2 veces por semana para prevenir pelos enquistados. Loción hidratante todos los días. Para rasurar usar gel de afeitar en dirección del pelo. Después aplicar aceite de rosa mosqueta o loción calmante.

== CUELLO ESCOTE CUIDADO ENVEJECIMIENTO ==
El cuello y escote envejecen tan rápido como el rostro pero se olvidan. Aplicar sérum y crema del rostro también en cuello y escote. Protector solar en estas áreas siempre. Movimientos de abajo hacia arriba en el cuello.

== MAQUILLAJE DURA EVENTO BODA DIA ENTERO ==
Para maquillaje de larga duración: piel hidratada, primer adecuado, base de larga duración, polvo fijador, primer de ojos para sombras, gel de cejas, dos capas de spray fijador. Llevar polvos y labial para retoques.

== LABIOS ANTES DE LABIAL PREPARAR ==
Antes de aplicar labial: exfoliar labios, aplicar bálsamo hidratante, esperar que se absorba, usar lápiz delineador de labios del mismo tono para que dure más y se vean más definidos, luego aplicar el labial.

== PERFUME FRAGANCIA COMO APLICAR DURAR MAS ==
Aplicar perfume en los puntos de pulso: muñecas, detrás de las orejas, en el cuello y detrás de las rodillas. No frotar las muñecas después de aplicar. Aplicar sobre piel hidratada o sobre ropa para mayor duración. Guardar en lugar fresco y oscuro.

== UNAS CUIDADO MANICURE COMO FORTALECER ==
Para uñas fuertes: usar base endurecedora, hidratar las cutículas con aceite de jojoba o almendras, evitar uñas en remojo prolongado, llevar una dieta rica en biotina y proteínas. Limar siempre en una sola dirección para no debilitarlas.

== CABELLO CUIDADO TIPOS RUTINA ==
Para cabello seco: shampoo hidratante con aceites, acondicionador nutritivo, mascarilla semanal, protector térmico antes de plancha o secadora. Para cabello graso: shampoo sin sulfatos agresivos, acondicionador solo en las puntas, no lavar con agua muy caliente que activa las glándulas sebáceas.

== PIEL CUELLO ESCOTE MANCHAS DECOLORAR ==
Para manchas en cuello y escote usar exfoliante suave dos veces por semana, sérum con vitamina C o ácido kójico y protector solar de día. Evitar ropa que roce constantemente y perfumes directamente sobre esa zona porque pueden causar manchas por el sol.
